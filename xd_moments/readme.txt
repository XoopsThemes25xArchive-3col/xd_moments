In the Zip file you will find the following elements:

1. PSD: photoshop source of the logo and slogan text.

2. Fonts: letter types used to write the logo name and slogan so you can edit them to your needs.

2. XOOPS Theme: the theme files, the folder xd_moments should be placed inside the themes folder of your xoops, then activate the theme from the admin side in 

system > general settings > preferences
